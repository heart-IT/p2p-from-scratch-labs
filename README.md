# P2P from Scratch — labs

Runnable companion labs for the [P2P from Scratch](https://heartit.tech) series.
Real Holepunch-stack software, small enough to read in one sitting — that's
the point.

| Lab | One command | Teaches |
|---|---|---|
| [`p2p-hello`](./p2p-hello) | `npx @heart-it/p2p-hello <passphrase>` | DHT discovery, hole-punching, encrypted transport |

Each lab is dual-runtime (Node today via `npx`; Bare/Pear-ready via
conditional imports) and carries its own README with the maintainer's
publishing checklist.

No servers, no accounts. Kill a lab and nothing remains.
